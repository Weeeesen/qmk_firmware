# ortho7x10 handwired

Custom handwired ortho7x10 keyboard.

* Keyboard Maintainer: [weeeesen](https://github.com/Weeeesen)
* Hardware Supported: Custom handwired 7x10 ortho keyboard (actual physical layout is 5x15)
